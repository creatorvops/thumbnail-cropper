# BatchCrop Pro — Chrome Extension + Web App

A powerful batch thumbnail generator designed for digital sellers on Etsy, Gumroad, Shopify, and more.  
Includes:

- Drag‑and‑drop batch image importing  
- 1:1 Etsy and 4:3 Gumroad safe‑area outlines  
- Diagonal repeated watermark pattern (anti‑theft)  
- Padding, background color, and aspect ratio controls  
- ZIP export using JSZip  
- Right‑click “Crop this image” Chrome extension support  
- Gumroad upgrade banner  

---

## 🚀 Features

### 🔹 1. Batch Image Processing
Upload multiple images at once and instantly generate thumbnails.

### 🔹 2. Safe‑Area Guides
Visual outlines show how Etsy and Gumroad crop thumbnails.

### 🔹 3. Anti‑Theft Watermark
Repeated diagonal watermark text across every image.

### 🔹 4. ZIP Export
Download all processed thumbnails in one ZIP file.

### 🔹 5. Chrome Extension Right‑Click Support
Right‑click any image → **Crop this image**  
Right‑click anywhere → **Open Thumbnail Cropper**
